# Citizen Appeals Visualizer

Одностраничное приложение для визуализации обращений граждан.  
Стэк: **React, Vite, JavaScript, MUI, React-Leaflet**.

## Функциональность

- Таблица обращений с пагинацией (10 на страницу)
- Фильтр по статусу
- Поиск по категории и адресу
- Карта с маркерами
- Модальное окно с подробной карточкой
- Адаптивность (Desktop + Tablet)

## Запуск

```bash
npm install
npm run dev
